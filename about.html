About our project

Our website serves a collection of photos of outer space (from Pexels) and each image is a link pointing to a page with more information about the picture. There, you can find details about what the image is and a link to learn more about the photographer.

Building the project

This website was built using Eleventy. Eleventy is a static site generator that supports templating, layouts, and more.
